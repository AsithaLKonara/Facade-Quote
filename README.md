# FACADE CENTER | Smart Software Quotation Platform

A premium, AI-powered estimation engine designed for high-end software development agencies. This platform specializes in the Sri Lankan ICT sector, providing deterministic, role-based pricing for both local and international (USD) engagements.

## 🚀 Key Features

- **✨ AI Architect Estimator**: Describe your project in natural language, and the system (powered by Llama 3.3) will architect the modules, assign roles, and estimate hours.
- **📊 4-Layer Financial Model**: Professional-grade pricing including Direct Delivery, Overhead (15%), Risk Reserve (10-25%), and Gross Margin (35%).
- **🌍 Multi-Currency Support**: Switch between LKR for local projects and USD for export clients with automated exchange rate conversion.
- **👨‍💻 Role-Based Estimation**: Price based on seniority bands (Architect, Senior, Mid, Junior, QA, DevOps) rather than generic tech rates.
- **📄 Professional Proposals**: Generate detailed software proposals with financial breakdowns and engagement terms.
- **🌱 Packaging Strategy**: Pre-defined tiers for Startup/MVP, Growth, and Enterprise projects.

## 🏗 Pricing Philosophy

FACADE CENTER uses a deterministic **Gross Margin** model:

1. **Delivery Cost** = Σ(Role Hours × Role Rate × Complexity)
2. **Overhead** = Max(Delivery × 15%, LKR 25,000)
3. **Risk Reserve** = Delivery × Risk %
4. **Final Price** = (Delivery + Overhead + Risk + Pass-through) / (1 - Target Margin)

## 🚦 Getting Started

### Prerequisites
- Node.js 18+
- Groq API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AsithaLKonara/Facade-Quote.git
   cd Facade-Quote
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment:
   Create a `.env.local` file in the root:
   ```env
   GROQ_API_KEY=your_key_here
   ```

4. Start development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) to start architecting.

## 📁 Project Structure

- `/app`: Next.js pages and API routes.
- `/components`: Reusable UI components (Quotation Print, Layout).
- `/data`: Pricing strategies, service catalogs, and rate cards.
- `/lib`: Core calculation engine and utility functions.
- `/types`: TypeScript interfaces for the domain model.
